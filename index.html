MIDI2LR

MIDI2LR is an application that translates MIDI commands into Lightroom actions. The wiki is read-only to prevent spamming. Contact me if you wish to add any pages or text. The wiki lists commands currently available through MIDI2LR. Instructions for use are in the wiki.

Latest Release Download

You can always download the latest release from https://github.com/rsjaffe/MIDI2LR/releases/latest. The wiki has a detailed listing of currently available commands.

Supported platforms

Versions 5+ of MIDI2LR support Lightroom Classic 11+. Version 4 supports Lightroom Classic 10. Use version 3 for earlier versions of Lightroom. Both macOS and Windows are supported, including macOS on Apple silicon (ARM) processors.
